Freibadstatus – Admin-Test

Diese Testversion speichert den Status lokal im Browser. Für die echte Nutzung würde dieselbe Oberfläche an eine kleine Datei oder Mini-Web-App gekoppelt, die ihr dann auf eurer Homepage ladet.

Nur Demo-Schutz. In echt wäre ein geschützter Link oder ein kleines Login sinnvoll.

Gemeinsamer Testspeicher: localStorage im Browser. Wenn du Admin und Widget im gleichen Browser öffnest, sieht man die Änderungen direkt nach dem Neuladen.

Vorschau

So testest du es

1. PIN 1234 eingeben.
2. Status und Text setzen.
3. Speichern klicken.
4. Danach die öffentliche Widget-Datei öffnen.
